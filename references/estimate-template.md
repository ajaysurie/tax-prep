# Tax Estimate Template

A rough, mechanical tax estimate included in the CPA package for reference only. This is NOT tax advice — it's a directional calculation to help the user and CPA see where things stand before the CPA does the actual return.

---

## Disclaimer (MUST include)

"This estimate is a rough directional calculation based on the data collected. It is NOT tax advice and should NOT be used for filing. Your CPA will produce the actual return with proper treatment of all items. This estimate does not account for: AMT, NIIT thresholds, passive activity limitations, QBI deduction phase-outs, or other complex interactions."

---

## Required Sections

The estimate should walk through these sections in order:
1. **Income summary** — all sources with amounts and source documents
2. **Adjustments** — HSA, SE tax deduction, student loan interest, IRA
3. **Deductions** — standard vs. itemized comparison
4. **Estimated tax** — federal (from brackets) + self-employment + state
5. **Payments and credits** — withholding, estimated payments, child tax credit
6. **Bottom line** — estimated balance due or refund

---

## Critical Rules

- When a line item has no source document yet, show "TBD — [document name] not yet received" instead of $0
- Label every number with its source document
- If data is incomplete, clearly state which documents are still missing and how that affects the estimate

---

## Calculation Notes

- Use brackets from `references/tax-data-sources.md` for the federal calculation
- State estimates: apply flat rate or simplified bracket from state guide
- Self-employment tax: net SE income x 92.35% x 15.3%
